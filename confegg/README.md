ConfEgg - Egg config library
============================

Usage
-----

*Python*
  * copy the `confegg/app.example.json` file to `confegg/app.json` and modify appropriately
  * add the app root to the PYTHONPATH (use the `pythonpath.sh` script)
  * `import confegg`
  * `config = confegg.get_config()`
